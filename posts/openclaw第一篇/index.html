養龍蝦一週記：OpenClaw 入門心得

2026/01/08 · 技術 · 閱讀約 1 分鐘

玩 Openclaw 養龍蝦也快一個星期了，已經花了 100 美在 API 上，還有一台 Mac mini，完全是富養小孩 😭

有點好奇詢問大家養龍蝦都用來做什麼呢？我大部分看到都是做個人的每日行程網站，或是自己的日記等等，做了不少 APP 或 Skill 應用，所以想知道有沒有人有一些瘋狂又酷炫的做法？

我自己用 openclaw 做了一團 AI Agent C-level 等，每個 Agent 都有不同的功能，舉例我可以用 Telegram 傳開會錄音檔給 AI，過一陣子他就會用 Notebooklm 幫我用成 Podcast 跟 簡報 與 會議記錄，並傳音檔或連結給我，這邊包成 skill 但還在優化，

這讓我很好去總結會議，之後想要接到 Linear 上統一管理 todo，SSOT 會做一份在 loacl 上管理，這樣不同 agent 就有對齊的標準。

另外 CTO 幫我管了 Claude code 跟 Lovable, Devin 等，還在想這樣的 multi agent 架構怎麼優化會比較好，看了一下 openclaw 的哲學，看起來需要稍微修改一下才能更好實作。

還有超多想搞的東西，之後在跟大家分享，就是 API 太燒了呀～ 有沒有什麼贊助商呀 🤣

想問問大家怎麼養龍蝦，是窮養還是富養？
