養蝦育兒須知-工具篇

2026/03/26 · 技術 · 閱讀約 3 分鐘

玩龍蝦也一陣子了，試過各種的工具跟 Skill，其實蠻多的，而且有很多東西都很方便。那真的要推的話，我覺得新手必裝的是以下四個，順便再偷渡推薦一下我自己的省錢 Skill 😂

𝟏. 本地部署 SearXNG（免費上網）

SearXNG 是一個開源的 metasearch engine，本地部署之後可以直接給龍蝦當搜尋引擎用。最大的好處就是完全免費，不用申請 Google API key 也不用付搜尋費用，龍蝦要查什麼就查什麼，不用擔心帳單。

部署方式蠻簡單的，Docker 跑起來就好，設定好 JSON output 讓龍蝦能解析搜尋結果，基本上設一次之後就不用再管它了。

𝟐. Agent Browser（省 token）

這個我覺得蠻多人忽略的。龍蝦內建的 Agent Browser 可以讓它直接操作瀏覽器，開網頁、點按鈕、填表單、截圖都可以。

為什麼說省 token？因為有些任務如果純靠 LLM 去解析網頁內容，要把整頁 HTML 塞進 context 裡面，token 用量超級大。但如果讓龍蝦直接用瀏覽器去操作，它只需要處理看到的畫面資訊，省下來的 token 其實很可觀。

記得用一個獨立的 browser profile，不要讓龍蝦碰到你自己的登入狀態。

𝟑. memory-lanced（變聰明）

龍蝦本身就有記憶系統，但 memory-lanced 這個 plugin 可以進一步增強它的記憶能力，讓向量搜尋更精準，記憶的召回跟儲存都更有效率。

簡單講就是龍蝦會更記得你跟它講過什麼，之前犯過的錯也會學起來，跟它共事的時間越長，它就越好用。我自己覺得這個裝了之後差異蠻明顯的，回答的品質跟上下文的連貫性都有提升。

𝟒. SecureClaw（安全）

這個真的是裝了安心。SecureClaw 是 Adversa AI 做的開源安全 plugin，它會幫你的龍蝦做安全稽核跟防護，包含 55 項檢查，涵蓋 prompt injection、credential 外洩、惡意 Skill 掃描等等。

老實說龍蝦能做的事情太多了，能讀寫檔案、跑指令、上網，如果沒有基本的安全防護，風險其實不小。尤其是從 ClawHub 裝 Skill 的時候，你不一定知道那個 Skill 裡面藏了什麼。SecureClaw 的 Skill scanner 可以幫你掃描惡意 pattern，用起來比較安心。

𝟓. skill-openclaw-map（省錢，自己的 Skill 偷渡推一下）

這個之前有分享過，就是我寫的一個 Skill，讓 Antigravity、Claude Code 這類 Coding agent 能直接維修跟精進你的龍蝦。

概念很簡單，Coding agent 不懂 OpenClaw 的架構，所以這個 Skill 裡面包了教它怎麼操作 OpenClaw、Log 在哪裡看、怎麼排 cron、怎麼修改你自己寫的 Skill 等等。這樣你的龍蝦掛了或是要調整 Skill，開個 Antigravity 搭配這個 Skill 就好，不用每次都燒 Opus 的 token，省很多。

我自己實測一天大概可以省 25 美金左右，維修全程都是 Opus 等級在幫你跑，但走的是 Coding agent 的額度，不是龍蝦的 API 帳單。

以上五個是我自己用下來覺得新手最值得裝的，裝完基本上養蝦體驗會好很多。

也歡迎大家補充，真的都有蠻多好用的東西，養龍蝦就是這樣，社群裡互相分享才會越來越順 🙌
