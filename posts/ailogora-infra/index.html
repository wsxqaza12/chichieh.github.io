AILogora 上線了：技術棧與踩坑記錄

2025/09/26 · 技術 · 閱讀約 1 分鐘

上週五 AILogora 上線了，也同步寄出了邀請信，總算在最後趕上線，過程遇到滿多問題的，坑也算踩好踩滿，紀錄一下目前用的技術棧，原則就是能蹭就蹭 🤣 以往不會自己走過這麼多面向，都是負責其中一部份，像是模型 server、後端或資料處理等，現在很能體會各位全端的辛勞呀。

⚙️Infra

  • Server：EC2
  • Web Server / Reverse Proxy：Nginx
  • Domain & DNS：Cloudflare
  • Storage：S3
  • SSL：Let’s Encrypt + Certbot

🧑‍💻Application Layer

  • Framework：Next.js (Bun Runtime)
  • Database & Auth：Supabase

📬Communication

  • Email：Zoho
  • Transactional：Resend + Brevo (兩邊蹭，之後規劃轉向 SES)

🔄DevOps

  • Version Control：GitHub
  • CI/CD：GitHub Actions

大家有沒有甚麼更方便的方式，監控與 Logging 正在規劃中，有什麼好建議嗎? SaaS 或自架都可。
